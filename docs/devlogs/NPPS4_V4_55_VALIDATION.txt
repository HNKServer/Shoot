- Finalize the CN home announcement carousel and WebView banner assets.
- Remove the abandoned attempt to transplant the international Museum catalogue into the CN client.
- Preserve native CN Museum behavior and the original CN download/package sequence.
- Keep Android and PC Python service trees synchronized.

Home announcement banner
------------------------
- Front item 1 remains the data-transfer WebView page.
- Remaining front items are active type-1 scouting banners derived from server_data.json.
- The back side remains the official manga type-2 WebView page.
- Data-transfer and manga thumbnails are packaged as dedicated server assets, not inserted into client ZIP packages.
- The server resolves both the declared PNG paths and the CN client's appended `.imag` cache-key paths.
- /download/getUrl reports the actual bundled image byte size.
- /cn-extracted serves the same PNG bytes with image/png for both plain and `.imag` aliases.
- Transfer image: 1914x616, RGBA, four outer corner pixels transparent.
- Manga image: 623x241.

Download-chain containment
--------------------------
- Client/content version remains 97.4.6.
- android_extra_update_packages defaults to [].
- No 99_0_116 or 99_0_117 is generated or registered.
- No normal/full data package is modified, re-packed, or size-adjusted.
- Operator-owned CN ZIP archives remain read-only.
- Old NPPS4-generated Museum overlay files/directories are removed from the server workspace during backend initialization.

Museum cleanup
--------------
Removed from both Android and PC source trees:
- npps4/system/museum_bridge.py
- npps4/tools/cn_museum_bridge.py
- npps4/assets/cn_museum_bridge/
- tools/cn_museum_bridge_toolkit/
- merged Museum client DB/Lua generation and content-package overlay logic
- Museum bridge config fields, diagnostics, and automatic login-bonus synchronization

Retained behavior:
- /museum/info remains the native NPPS4/CN route.
- CN uses the bundled native honoka_main.db Museum master with 16 rows.
- Existing user unlocks are filtered to IDs present in that native master.
- Legacy automatic cross-region grant records and non-native unlock IDs are pruned, while legitimate native unlocks are preserved.
- Existing configs are migrated by removing obsolete Museum bridge options and NPPS4-generated 116/117 entries; a backup is created when a change is made.

Android editor regression guard
-------------------------------
- Draggable vertical and horizontal scrollbars remain present.
- Dragging text/scrollbars does not force the keyboard open.
- First stationary tap focuses and positions the cursor.
- A second deliberate stationary tap opens the soft keyboard.

# Android Wrapper v4.3

- Rebased on the native Pydantic v2 profile.
- Does not use the Pydantic v1 shim or compatibility monkey patches.
- Adds reproducible scripts to build `pydantic-core` Android wheels for Chaquopy.
- Adds `constraints-android.txt`, generated by the wheel builder, to keep
  `pydantic` and `pydantic-core` versions aligned.
- Adds `scripts/build_apk_pydantic2.*`, which refuses to build until the required
  `pydantic-core` wheel exists.
- Keeps Python 3.13 to satisfy NPPS4's Python >= 3.12 requirement while matching
  modern Android wheel tooling.
